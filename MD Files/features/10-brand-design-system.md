# Feature: Progress Design System

The app must use the visual standards in `design/progress-design-system.md`.

## Key requirements

- Primary buttons are red `#EB0249`.
- Secondary buttons are blue `#054BFF`.
- Body text is `#383F55`.
- Page background is `#F4F7FD`.
- Card borders are `#DDE9F7`.
- Form borders use `rgba(158,174,195,0.80)`.
- Table headers use `#EEF5FF`.
- Cards use 10px radius and Progress shadows.
- Buttons are uppercase and 43px high by default.
- No separate hotfix CSS files.
