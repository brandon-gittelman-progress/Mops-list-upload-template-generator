# Feature: Step 3 Data Confirmation

Step 3 is conditional and always represented in the 6-step wizard.

## Confirmation cards

- Required partial columns.
- MAL/MQL detection.
- State blanking notices.

## Rules

- Partial confirmation applies only to required standard export fields.
- Optional fields do not block the user.
- Kept fields do not block the user.
- `Lead Source - Initial` is excluded.
- State blanking notices are informational.
