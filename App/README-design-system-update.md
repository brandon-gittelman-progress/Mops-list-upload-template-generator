# Progress Design System Update

Replace the existing CSS files in `progress-list-upload/styles/` with these files:

- `tokens.css`
- `base.css`
- `components.css`
- `steps.css`

No new CSS files are required.

## Important behavior preserved

This is a design-system update only. It does not change JavaScript behavior, validation, export schema, Teams integration, or app flow.

## Design system changes applied

- Progress primitive and semantic color tokens from the uploaded guide.
- Primary CTA buttons use Progress red #EB0249 with #B90138 hover.
- Secondary buttons use Progress blue #054BFF with #0037CB hover.
- Body text uses #383F55; headings use #060606.
- Page background uses #F4F7FD.
- Tables use #EEF5FF sticky headers and zebra striping.
- Inputs use rgba(158,174,195,0.80) borders and 5px radius.
- Cards use 10px radius and Progress shadow tokens.
- Buttons are all-caps and use 43px default medium height.
- Spacing uses the Progress px scale and 30px gutters.
