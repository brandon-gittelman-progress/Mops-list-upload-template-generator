# Progress Brand CSS Update

Replace the existing files in `progress-list-upload/styles/` with the files in this ZIP:

- `tokens.css`
- `base.css`
- `components.css`
- `steps.css`

No new CSS file is required. These updates integrate the Progress brand system into the existing stylesheet structure.

## What changed

- Updated color tokens to Progress Navy, Green, Surface Gray, Action Blue, Alert Red, and DarkGreyS4.
- Updated typography tokens to Progress Sans Display/Text with Helvetica Neue/Arial fallbacks.
- Updated spacing to Progress increments and 30px gutters.
- Updated buttons to Progress sizing and colors.
- Updated inputs to #fcfdfe background, #9EAEC3 border, 4px radius.
- Updated cards to 8px/10px radius and approved shadows.
- Updated Step 5 issue/resolved colors to use Progress brand colors.
